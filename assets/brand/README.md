# Kith brand assets

Logo and promotional images for the project. All marks use the peer-to-peer
"circle of equal nodes" motif.

| File | Size | Background | Use |
| ---- | ---- | ---------- | --- |
| `kith-icon-1024.png` | 1024×1024 | transparent | App / favicon source, square icon mark |
| `kith-logo-lockup-2400x720.png` | 2400×720 | transparent | Horizontal icon + wordmark lockup |
| `kith-readme-banner-1280x640.png` | 1280×640 | designed | README header and GitHub social preview |

## Palette

| Role | Hex |
| ---- | --- |
| Base (deep slate/indigo) | `#0E1B2E` |
| Primary accent (teal) | `#2DD4BF` |
| Secondary accent (amber) | `#F5A623` |

## Notes

- The banner doubles as the repository **social preview** image
  (Settings → General → Social preview).
- Keep the wordmark spelled exactly **Kith**.
- Prefer the transparent PNGs over re-rendering when embedding the mark elsewhere.
