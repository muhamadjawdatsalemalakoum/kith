How it works

A flat mesh: every device is an equal peer holding a full, encrypted replica. No hub, no data-bearing server.

Install & open

Download Kith on each computer you own. It just works — there's nothing to sign up for.

Link with a code

Show a one-time code on one device, paste it on the other. They derive a shared key with SPAKE2 and never send the code over the wire.

Stay in sync

Changes flow directly between your devices over encrypted QUIC. Offline edits are pending, not lost.

Built for your AI, too

Kith speaks MCP. Point Claude Desktop or Cursor at it and your assistant works with your own memory, tabs, and files — locally, with no server in the middle.

Add to your MCP client: 
{ "mcpServers": { "kith": { "command": "/path/to/kith", "args": ["serve"] } } }
memory.appendmemory.searchmemory.read tabs.addtabs.list files.sharefiles.fetch
